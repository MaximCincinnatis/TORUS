Open Browser Console (F12) then Refresh http://localhost:3000

Look for these console logs:

  • 📊 Calculating supply projection from day X to Y
  • 🔍 Processing days X to Y (starting from current protocol day)
  • 📊 BEFORE filtering: First projection day: X
  • 📊 Filtering from day 38 forward
  • 📊 AFTER filtering: First day shown: X

Expected:

📊 AFTER filtering: First day shown: 38

If it shows 39:

The projections array doesn't include day 38, possibly because:

  1. No reward data for day 38 (causes continue/skip)
  2. Projection array starts from wrong index
  3. Filter is excluding day 38
